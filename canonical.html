(Tiny) Group Introduction

A permutation is a bijective function on $\{1,\dots,n\}$

Permutations can be written as $(1,3,5)$

A group is a set of permutations closed under composition and inverse

(Tiny) Group Introduction

Given a set $S = \{s_1,\dots,s_n\}$ and a permutation $p$, then: $S^p = \{{s_1}^p,\dots,{s_n}^p\}$.

Example: $\{2,4\}^{(2,3,1)} = \{3,4\}$

Problem

Given:

  • Group $G$
  • Two subsets $S, T \subseteq \{1,\dots,n\}$

Is there a $g \in G$ such that $S^g = T$?

Example 1

Given:

  • Group $C_5 = \langle (1,2,3,4,5)\rangle$
  • $S = \{1,3\}$, $T = \{3,5\}$

$S^{(1,3,5,2,4)} = T$

Example 2

Given:

  • $S = \{1,3\}$, $T = \{3,4\}$

Nothing maps $S$ to $T$

More Sets

Given:

  • Group $G$
  • Three subsets $S, T, U \subseteq \{1,\dots,n\}$

If $S^{g_1} = T$ and $T^{g_2}=U$, then

    $S^{g_1.g_2} = U$

Minimal Image

Given a group $G$, object $x$ then:

    $Min(G,x) = Min\{x^g | g \in G\}$

for sets, we define:
$S < T$ if $Min(S-T) < Min(T-S)$

Give the permutation

Why Minimal Image?

Given group $G$, sets $S, T$:

  • $\exists g \in G. S^g = T$ iff
  • $Min(G, S) = Min(G,T)$

Finding equivalence classes

So, we can run $Min$ on every set, and put sets together with the same minimum.

But we do not really need Min

Beyond Minimal Image

A function $Can(G,X)$ is a canonicalising function if:

  • $\exists g \in G. Can(G,X) = X^g$
  • $\forall g \in G. Can(G,X) = Can(G,X^g)$

Other problems

Find if $Can(G,X) \leq Can(G,Y)$

Implementing Minimal Image

  • Split G into a disjoint union: $\{S_1,\dots,S_n\}$
  • Then: $Min(G,X)=$
      • $min\{Min(S_1,X), \dots ,Min(S_n,X)\}$
  • Make $S_1$ a subgroup of $G$, and other $S_i$ its cosets
  • Then $Min(G,X)=$
      • $min\{M(S_1,X), M(p_1.S_1,X),\dots\}$
      $min\{M(S_1,X), M(S_1, X^{p_1^{-1}}), \dots\}$
Scary maths

Implementing Minimal Image

  • Split as: $Min(G,X)=$
      • $min\{Min(H,X_1), \dots ,Min(H,X_n)\}$

Example

$Min(Sym(5), \{2,3\})$ is the smallest of:

  • $Min(Sym(\{2\dots 5\}), \{2,3\})$
  • $Min(Sym(\{2\dots 5\}), \{1,2\})$
  • $Min(Sym(\{2\dots 5\}), \{1,5\})$
  • $Min(Sym(\{2\dots 5\}), \{4,5\})$
  • $Min(Sym(\{2\dots 5\}), \{3,4\})$

Ordering algorithm

Given a group, we order integers as:

  • First put all integers which are fixed
  • Pick a point from the smallest orbit, fix that
    • Loop until all integers put in order

Sets of Sets

Given two sets of sets $S$ and $T$, $S <_3 T$:

  • If a unique value occurs 3 times in both $S$ and $T$, and that value is smaller in $S$.
  • Otherwise, $S < T$ under traditional set of set ordering.
